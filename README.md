# Buzzer

## Utilisation

1. Install `socket.io`:
  ```
  npm install
  ```
2. Start server (default port is 4000):
  ```
  node index.js
  ```
3. Les differents pages
  ```
  http://<server-ip>:4000/host.html
  http://<server-ip>:4000/user.html
  http://<server-ip>:4000/board.html
  ```
